# TripMate

TripMate is a travel planning web app for turning loose ideas into practical, shareable trips.

## Stack

- Next.js App Router
- React
- TypeScript
- CSS Modules / global CSS to start, with room to add a design system later

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Codex Setup

- `AGENTS.md`: project-wide instructions for Codex agents
- `.codex/agents/tripmate-product-engineer.md`: suggested agent role for TripMate work
- `.codex/skills/tripmate/SKILL.md`: TripMate-specific product and domain skill

## Product Direction

TripMate should help travelers collect destinations, compare options, build day-by-day plans, track budget and logistics, and collaborate with travel companions.
